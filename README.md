## ✨ Funcionalidades

- **🚀 Design Moderno**: Interface limpa e sofisticada inspirada em estética premium.
- **📱 Totalmente Responsivo**: Otimizado para todos os dispositivos (Desktop, Tablet e Mobile).
- **🎭 Animações Suaves**: Transições e efeitos de entrada utilizando Framer Motion.
- **📁 Showcase de Projetos**: Carrossel interativo para exibição de trabalhos recentes.
- **💬 Contato Facilitado**: Integração com formulário Web3Forms e botão flutuante de WhatsApp.
- **⚡ Performance Extrema**: Construído com Vite para um carregamento instantâneo.

## 🚀 Como Iniciar

### Pré-requisitos
- [Node.js](https://nodejs.org/) (v18 ou superior)
- npm ou yarn

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/portfolio.git
   ```

2. Entre no diretório do projeto:
   ```bash
   cd portfolio
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Configure as variáveis de ambiente:
   - Renomeie o arquivo `.env.example` para `.env`.
   - Adicione sua chave do Web3Forms e seu número de WhatsApp.

5. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

6. O projeto estará rodando em `http://localhost:5173`.

---

## 🔑 Variáveis de Ambiente

Para que o formulário de contato e o botão de WhatsApp funcionem corretamente, você precisará configurar as seguintes variáveis no seu arquivo `.env`:

- `VITE_WEB3FORMS_ACCESS_KEY`: Sua chave de acesso do Web3Forms.
- `VITE_WHATSAPP_NUMBER`: Seu número de WhatsApp (com DDI e DDD).
